# GoEDR
#### Simple single-host EDR written in Go

### Functionalities
1. Checks temp, downloads, desktop, document folders against a list of malicious file hashes
2. Tests network connections against list of malicious IPs
3. Runs daily scans

### Setup
1. npm run build
2. go build main.go
3. ./main
